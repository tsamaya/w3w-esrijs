what3words addressing the world with ArcGIS API for Javascript. This application demonstrates integration of ArcGIS API for JavaScript API with what3words API; Open source, Apache 2.0 licensed, and available on GitHub.

Made in Grenoble by Arnaud Ferrand, July 2015

Features
  • what3words on map click
  • what3words on geocoding addresses
  • position with a what3words (word.word.word) in search input
  • what3words on device locate
